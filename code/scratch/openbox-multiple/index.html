Open the Box

  1. Create an ERC20 token to be received as a reward from opening the box (BoxToken)
  2. Create an ERC721 token that's required to open a box. Different ERC721 tokens can open different boxes (KeyNFT)
  3. Create an ERC721 token for each box (BoxNFT)
  4. Boxes can be opened and closed, and so need open/closed state variable
